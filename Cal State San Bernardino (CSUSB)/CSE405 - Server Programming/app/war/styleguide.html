Notes on Styleguides

  • Using 2 Spaces for Indent: I thought I had my spacing and indentation in order, but upon taking the time to fix it using 2 spaces, I was shocked at how clean the code looked
  • Used protocol-less url's for loading external files
  • Stopped using onclick for my buttons and started using proper HTML semantics
  • Separted any style elements from the HTML document and used .css files
  • Separted code blocks better than I previously had it
  • Ran the HTML Validator and the CSS Validator and removed errors
  • Removed optional tags: noticed I had setup id's but never used them, so I removed them
  • I only have one CSS element but fixed spacing on it
  • Cleaned up javascript using better spacing and separting code blocks with blank lines
  • Overall, the style guides helped make the code look cleaner and easier to look at. Will probably make it easier in the future if I need to change anything
